This plugin provides the way of using interactive mode from other process via TCP.

(Quick Reference)

improxClient.groovy

Purpose

The improxClient.groovy script invokes a specified command on the interactive mode proxy server via TCP port.

Examples

groovy improxClient.groovy help
groovy improxClient.groovy test-app unit: sample.SampleUnitTests

Description

Usage:

groovy [-Dimprox.port=PORT] improxClient.groovy COMMAND_LINE

Arguments:

  • COMMAND_LINE - The command line which you want to execute

System properties:

  • improx.port (optional) - The port number which is used for connection between client and server (default: 8096)
